## ‚ö° Features

### 1. Multi-Protocol Support (Neural Links)
Generate specialized QR codes for various data types:
* **Neural Link (URL):** Direct web navigation.
* **WiFi Access:** Encrypted network credentials (SSID/Password/Encryption).
* **Identity Card (vCard):** Digital business cards with Name, Phone, and Email.
* **Encrypted Text:** Raw string data for secure messaging.

### 2. Deep Customization
* **Dynamic Styling:** Choose between **BLOCK** (Classic Square) or **CELL** (Rounded) dot patterns.
* **IDENT System:** Auto-fetches favicons from URLs to embed as logos in the center of the QR code.
* **Memory Upload:** Overlay the QR code on top of custom background images with adjustable alpha transparency.
* **Data String Toggle:** Optionally render the data label directly onto the QR asset.

### 3. Extraction & Sync
* **Live Rendering:** Real-time QR generation as you type or change settings.
* **Data Extraction:** One-click "Extract" button to download high-resolution PNG files.
